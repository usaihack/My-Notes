
### **Kali is a tree**:

| /                                                      | It is the opening point, like ground is the opening point for a tree. It is the top directory. Everything is inside it e.g.  `home  etc  var  bin  root`                                                                                                                                                                                                                                                                                          |
| ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| /home                                                  | Directory inside / which contains all users directories. e.g. `/home/usman` which means user `usman` in `home` exist                                                                                                                                                                                                                                                                                                                              |
| ~                                                      | Shortcut to a user's `home` e.g. shortcut to `/home/usman` will be `~`                                                                                                                                                                                                                                                                                                                                                                            |
| root                                                   | Like an admin in windows. Has unlimited power, can do anything                                                                                                                                                                                                                                                                                                                                                                                    |
| /root                                                  | Private directory for the `root`, home of `root`                                                                                                                                                                                                                                                                                                                                                                                                  |
| /etc<br><br>(Hackers are very interested in this path) | It acts as a rule book for users. It stores who can do what and when and how etc...<br><br>It has some important directories:<br><br>1. `/etc/passwd` => Users on the system (Who)<br>2. `/etc/shadow` => Hashed passwords (Who)<br>3. `/etc/sudoers` => Who can act as an Admin (means the user can do some tasks which only admin can do, but not as a `root` , with the help of `sudo`).<br><br>Simply say, it has all settings about kali OS. |
| /var                                                   | Stores variable (changing data) data, like logs in `/var/log`                                                                                                                                                                                                                                                                                                                                                                                     |
| /bin                                                   | Toolbox, all commands run from here like `ls`, `cat`, `cp`, `mv` etc.                                                                                                                                                                                                                                                                                                                                                                             |
| /tmp                                                   | Temporary files lives here which often cleaned on reboot. (Short memory)                                                                                                                                                                                                                                                                                                                                                                          |
