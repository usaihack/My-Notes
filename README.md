## 🚀 Getting Started

To explore these notes in the best possible format:

1. **Install Obsidian** - Download from [obsidian.md](https://obsidian.md/)
2. **Clone this repository**
   ```bash
   git clone https://github.com/usaihack/My-Notes.git
   ```
3. **Open in Obsidian** - Use "Open folder as vault" and select the cloned directory

### 📱 Mobile & Desktop Sync (Optional)

For seamless two-way synchronization across devices:

1. **Install MEGA** - Desktop and mobile apps from [mega.io](https://mega.io/)
2. **Setup Sync** - Configure MEGA to sync this repository folder
3. **Access Anywhere** - Your notes stay synchronized across all devices

For detailed setup instructions, see [SETUP.md](SETUP.md).
